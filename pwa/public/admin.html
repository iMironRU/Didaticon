⚠ Сервис не запустился за 60 секунд
Возможные причины:
  • Неверные параметры OIDC (ISSUER, CLIENT_ID, CLIENT_SECRET)
  • Ошибка подключения к базе данных
  • Не хватает переменных окружения в .env
Что делать:
  1. Посмотреть логи: docker logs eios-glue-1 --tail 50
  2. Исправить параметры выше и нажать «Сохранить»
  3. Нажать «⟳ Перезапустить сервис» снова
  4. Если не помогает — перезапустить вручную:
    docker compose restart glue
Эта страница продолжает работать — она обслуживается nginx независимо от glue.
